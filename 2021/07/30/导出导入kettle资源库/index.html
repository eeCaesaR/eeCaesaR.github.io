如何导出导入kettle资源库

1. 导出

1) 导出资源库的前提，首先要连接到相应资源库。点击右上角 Connect，然后登录进入相应的资源库

2) 登入成功后，进入 工具 –> 资源库 –> 导出资源库，即可导出当前资源库

2. 导入

1) 导入资源库前提，必须先配置好一个资源库！

2) 右上点击 Repository Manager

3) 选择 Other Repositories –> Database Repository

4) 然后建立一个数据库连接

5) 注意如果用 Mysql，jdbc 方式连接的话，必须要下载相应的 mysql driver class 驱动文件

到这个链接 https://dev.mysql.com/downloads/connector/j/ ，下载 5.1.49 版本

并将其中两个jar包：

mysql-connector-java-5.1.15-bin.jar

mysql-connector-java-5.1.49.jar

放到 kettle 安装目录的 /lib/ 目录下面

然后重启 kettle

6) 重启kettle 之后，先链接配置好的资源库，默认账密 admin/admin

7) 然后导入资源库即可
